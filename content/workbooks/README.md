# Workbook Templates

ARM deployment templates for Microsoft Sentinel workbooks (dashboards).

Each workbook provides interactive visualizations for monitoring SpyCloud data.

## Workbooks

| Workbook | Description |
|----------|-------------|
| Executive Summary | High-level exposure metrics and trends |
| SOC Operations | Day-to-day security operations dashboard |
| Identity Exposure | User-level exposure analysis |
| Breach Catalog | Breach source tracking and analysis |
| Connector Health | Data connector status monitoring |
| Conditional Access | CA policy response tracking |
| Compass | Application and device exposure |
| Investigations | Deep-dive investigation tracking |
| Remediation | Remediation progress and effectiveness |
| Graph Analysis | Identity relationship visualization |
| Threat Intelligence | Threat landscape overview |
| Defender CA Response | Defender and CA combined view |
