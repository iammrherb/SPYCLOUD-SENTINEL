✓ Prerequisites

  • ✓
    Azure Subscription with Microsoft Sentinel An active Azure subscription with a Log Analytics workspace and Microsoft Sentinel enabled on it.
  • ✓
    Contributor Role on the Resource Group The deploying user or service principal must have Contributor (or higher) permissions on the target resource group.
  • ✓
    SpyCloud API Key A valid API key from your SpyCloud portal. Contact spycloud.com to obtain enterprise API access.
  • ✓
    Microsoft Sentinel Contributor Role Required for deploying analytic rules, workbooks, and automation rules into the Sentinel workspace.
  • ✓
    Logic App Contributor (for Playbooks) Required if deploying automated response playbooks. The Logic App managed identity will need appropriate Graph API permissions.

⚡ Quick Start

  1. Click "Deploy to Azure"

    Use the button above to open the Azure Portal with the SpyCloud ARM template pre-loaded. The deployment wizard will guide you through configuration.

  2. Select Your Workspace

    Choose the resource group and Log Analytics workspace where Microsoft Sentinel is enabled. The template will create all required custom tables and data collection resources.

  3. Configure the SpyCloud API Key

    Enter your SpyCloud API key when prompted during ARM template deployment. After deployment, you must also enter the key on the Sentinel connector page (Data connectors → SpyCloud → Open connector page → Enter API key → Connect) to activate the data pollers.

  4. Review and Deploy

    Review the selected components (connectors, analytics, playbooks, workbooks) and click Create. Deployment typically completes in 5-10 minutes.

  5. Verify Data Ingestion

    After deployment, navigate to Sentinel > Data Connectors and verify the SpyCloud connector shows as Connected. Run SpyCloudBreachWatchlist_CL | take 10 to confirm data flow.

  6. Enable Analytic Rules

    Go to Sentinel > Analytics > Rule Templates, filter by "SpyCloud", and enable the rules relevant to your organization. Rules are deployed as templates so you control which are active.
